## 🚀 Featured Projects

### 🛡️ Bot-Shield — Cloud-Native DevSecOps Middleware
> **Node.js · AWS EC2 · CloudWatch · NPM**

[![NPM Downloads](https://img.shields.io/badge/NPM%20Downloads-550%2B-CB3837?style=flat-square&logo=npm&logoColor=white)](https://www.npmjs.com/package/bot-shield)
[![License](https://img.shields.io/badge/License-MIT-green.svg?style=flat-square)](LICENSE)
[![Open Source](https://img.shields.io/badge/Open%20Source-❤️-red?style=flat-square)](https://github.com)

An open-source security middleware engineered to mathematically detect and block automated scraping bots using **V8 cryptographic handshakes** — deployed globally on the NPM registry.

**Key Highlights:**
- 🔐 **V8 Crypto Handshakes** — cryptographic browser-fingerprinting to distinguish bots from humans
- 📊 **Real-Time Telemetry** — live CloudWatch dashboards for threat monitoring on AWS EC2
- 🔌 **Drop-in Integration** — designed for seamless adoption in any Node.js server
- 📦 **550+ Combined Downloads** across two major versioned releases

```
npm install bot-shield
```

### 🛒 Cybercart — Event-Driven Microservices Architecture
> **Python · Flask · AWS (SNS · SQS · DynamoDB) · Terraform · Docker**

[![IaC](https://img.shields.io/badge/IaC-Terraform-5835CC?style=flat-square&logo=terraform&logoColor=white)](https://terraform.io)
[![Containers](https://img.shields.io/badge/Containers-Docker%20Compose-0db7ed?style=flat-square&logo=docker&logoColor=white)](https://docker.com)
[![Uptime](https://img.shields.io/badge/Uptime-99.9%25-brightgreen?style=flat-square)](https://github.com)

A fully containerised **pub/sub microservice mesh** simulating a decoupled, high-throughput eCommerce checkout flow — provisioned entirely through Terraform.

**Key Highlights:**
- ☁️ **Full IaC** — entire cloud infrastructure (SNS topics, SQS queues, DynamoDB tables) provisioned via Terraform
- 🔄 **Event-Driven Pub/Sub** — decoupled services communicating asynchronously through AWS SNS → SQS fan-out
- ⚡ **DLQ Circuit Breakers** — automated Dead Letter Queue logic quarantines toxic payloads, ensuring **99.9% uptime**
- 🐳 **Fully Containerised** — Docker Compose orchestrates all microservices locally for a seamless dev-to-deploy lifecycle

```
Architecture: Flask API → SNS Topic → SQS Queues → Consumer Services → DynamoDB
                                               ↓ (on failure)
                                          Dead Letter Queue (DLQ)
```
